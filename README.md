# Ultrasonic-sensor-01

### Sketch for testing ultrasonic sensor with arduino.
Used to measure the distance of an object (printed on serial port). Turn on led if dstance above a value (5cm)

#### Devices:
* Arduino uno
* Ultrasonic Sensor HCSR04

##### Pin Connection:
Sensor | Arduino
------- | --------
Vcc | 5V
Echo | 10
Trig | 9
GND | GND
